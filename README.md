## C'est GitHub Pages ❓



GitHub Pages est un service gratuit proposé par GitHub qui permet de publier des sites web statiques directement depuis un dépôt GitHub.

Concrètement, il transforme les fichiers HTML, CSS et JavaScript présents dans ce dépôt en un véritable site web accessible depuis n'importe quel navigateur.

### Avantages

* ✅ Hébergement gratuit
* ✅ Déploiement automatique après chaque mise à jour
* ✅ HTTPS inclus
* ✅ Aucune publicité
* ✅ Idéal pour les portfolios étudiants et professionnels
* ✅ Partage facile avec enseignants, recruteurs et collaborateurs

## Mise en ligne avec GitHub Pages

1. Pousser le projet sur GitHub.
2. Ouvrir les paramètres du dépôt.
3. Accéder à la section **Pages**.
4. Dans **Build and deployment** :

   * Source : Deploy from a branch
   * Branch : `main`
   * Folder : `/ (root)`
5. Cliquer sur **Save**.
6. Attendre quelques minutes.

GitHub générera automatiquement l'URL publique du site.
